# **Gallery** is a simple website project built using **HTML** and **CSS**.
This project represents my **first attempt at creating a website** from scratch. 


## 📌 Description
The website functions as a basic gallery and was created as a way to practice HTML structure and CSS styling. Through this project, I aimed to understand how a web page is built and how HTML and CSS work together. 
I am aware that the project contains **some mistakes and non-optimal practices**, which is expected since I am still at the beginning of my journey in web development. 


## 🧠 What I Learned By working on this project, I learned: - The basic structure of an HTML document - How to use CSS for layout and styling - Core concepts such as: - divs and classes - positioning - flexbox / basic alignment - How to organize a simple front-end project - The basics of **Git and GitHub** 


## 🚀 Future Improvements In the future, I would like to: - Improve the code structure and readability - Make the website more responsive - Apply better CSS practices - Add JavaScript for interactivity 


## ℹ️ Note This project was created for **learning purposes only** and reflects my current skill level. Any feedback is welcome!
